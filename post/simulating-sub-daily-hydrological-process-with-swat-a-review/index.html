Simulating sub-daily hydrological process with SWAT: a review

2019-11-04 3 min read # Model # SWAT # Hydrology # Literature # SCI # Review

Ciation:

Tássia Mattos Brighenti, Nadia Bernardi Bonumá, Raghavan Srinivasan & Pedro Luiz Borges Chaffe (2019) Simulating sub-daily hydrological process with SWAT: a review, Hydrological Sciences Journal, 64:12, 1415-1423, DOI: 10.1080/02626667.2019.1642477

Keywords:

hydrological modeling; SWAT; sub-daily simulation; time step

Key points:

  • Using sub-daily data improves hydrograph peak simulation, while for medium flows use of daily data was better./日数据在中尺度应用更好，小时数据可以提高水文峰值模拟的精度
  • 1-hour time step was the most suitable time scale for the subdaily model application./1小时间隔的数据对于日尺度上的水文模拟应用是最好的
  • The main challenge for using the sub-daily routine was the lack of high-resolution data./日尺度的水文模拟研究最大的阻碍是缺乏高分辨率的数据

What I've Learned:

  • Main Conclusions
    • GAML method can mainly improve hydrograph peaks
    • CN gives good performance for medium flow
    • sub-daily data significantly improves the suspended sediment and nutrients modeling
    • sub-daily modules was better at simulation of the exported flood volume
    • the best time scale for sub-daily SWAT is the 1-hour interval
  • Useful Tips
    • model simulation can be judged as satisfactory if the Nash-Sutcliffe efficiency criterion (NS) > 0.50 and the ratio of the root mean square (RSR) < 0.70, and if the percent bias (Pbias) is ± 25% for streamflow, ± 55% for sediment, and ± 70% for nitrate and phosphorus for measured data of typical uncertainty.
    • For current SWAT applications, the Sequential Uncertainty Fitting (SUFI2) is the tool most used for sensitivity analysis and model calibration
    • The Green-Ampt Mein-Larson(GAML) method is used for sub-daily rainfall to calculate the infiltration as a function of the wetting front matric potential and effective hydraulic conductivity,while the Curve Number(CN) is used for daily rainfall/一般逐日数据采用CN方法计算入渗量与湿润前沿基质势和有效水力传导率的函数关系，子日数据则使用GMAL方法
