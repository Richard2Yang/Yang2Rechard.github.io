SWAT-CUP中需要注意的问题

2019-12-31 7 min read # Model # SWAT # Hydrology # Notes # SWAT-CUP # Calibration&Validation

一些重要的参数

衡量标准

  • 95PPU
    95% prediction uncertainty
    95%的置信区间(95％的预测不确定性)
  • P-factor
    P-factor is the percentage of observed data enveloped by our modeling result, the 95PPU, theoretically, this value is range between 0 and 100%.
    P因子是模拟结果所覆盖的观察数据的百分比(95PPU), 当95%以上的模拟值与实测值一致时, 表示模拟结果越好, 因此, 该值越大越好(理论上介于0-1之间)。
  • R-factor
    R-factor is the thickness of the 95PPU envelop, theoretically, this value is range between 0 and infinity.
    R因子是95%置信区间的数据的宽度, 该值越小说明模拟的结果越好(理论上介于0到∞\infty∞)。

No hard numbers exist for what these two factors should be, similar to the fact that no hard numbers exist for R2R^2R2 or NS. The larger they are, the better they are. For P-factor, we suggested avalue of >70% for discharge, while having R-factor of around 1. For sediment,a smaller P-factor and a larger R-factor could be acceptable.
对于径流模拟来说, P-factor>0.7且R-factor接近1是可以接受的, 对于沉积物而言, 较小的P-factor和较大的R-factor是可以接受的，没有硬性的规定, R2R^2R2和Nash-Sutcliff(NS)同样如此，值越大表明模拟结果越好。
注：如果最初将参数范围设置为等于最大物理意义范围，但仍无法找到包含任何或大部分数据的95PPU，则问题不在于参数校准上，必须重新检查概念模型。

影响地下水(ground water)的变量


- GWQMN
Threshold depth of water in the shallow aquifer required for return flow to occur (mm H2OH_2OH2​O), range from 0 to 5000.
The shallow aquifer contributes base flow to the main channel or reach within the subbasin. Base flow is allowed to enter the reach only if the amount of water stored in the shallow aquifer exceeds a threshold value specified by the user,
浅水层中的发生回流所需的水阈值深度（mm H2OH_2OH2​O），该值介于0-5000之间。浅水层中的水会以基流的形式进入河道，只有当浅水层中的出水量超过该值时，才会有基流产生并进入河道。即该值越小，进入河道的基流越多
- GW_REVAP
Groundwater "revap" coefficient.（地下水再蒸发系数）
随着GW_REVAP接近0，水从浅层含水层到根部区域的流动受到限制。随着GW_REVAP接近1，从浅层含水层到根区的转移速率接近潜在的蒸散速率，该值越大代表浅层含水层中的水进入非饱和带的能力越强。GW_REVAP的值应在0.02和0.20之间。
- REVAPMN
Threshold depth of water in the shallow aquifer for “revap” or percolation to the deep aquifer to occur (mm H2OH_2OH2​O), range from 0 to 500.
浅水层中水发生“渗透”或渗入深水层的阈值深度（mm H2OH_2OH2​O），该值介于0到500之间。
仅当浅层含水层中的水量等于或大于REVAPMN时，浅层含水层的水才会运动到非饱和带，因此，该值越大意味着浅层含水层中的水越难进入深水层。
- ALPHA_BF
Baseflow alpha factor (1/days), baseflow recession constant(αgw\alpha{gw}αgw)
地下水回归常数，能够直接指示地下水对径流补给的变化。在地下水响应快的陆地该值介于0.9-1.0, 响应慢的地区介于0.1-0.3之间。
- GW_DELAY
通过渗漏或旁通流穿过土壤剖面底部进入并流经包气带，之后成为浅层地下水补给。在水分离开土壤剖面到进入浅层含水层之间的时间延迟，取决于地下水面的深度以及包气带和地下水饱和带地质组分的水力性质。该值无法直接测量，延迟时间越小，渗漏进入浅层含水层的水量越多
- RCHRG_DP
Deep aquifer percoltion fraction, the fraction of percolation from the root zone which recharges the deep aquifer.The value for RCHRG DP should be between 0.0 and 1.0. 含水层渗漏系数，该值越小代表浅层含水层向深层地下水的下渗越小，该值介于0-1之间

影响地表径流(surface runoff)变量

  • CN2
    CSC curve number (SCS径流曲线数)，通过SCS径流曲线数方程建立径流量与土壤渗透率、土地利用和前期土壤水分条件之间的函数，从而计算出径流量。
    SCS curve number equation(SCS径流曲线数方程):

    Qsurf=(Rday−Ia)2(Rday−Ia+S)Q_{surf}=\frac{(R_{day}-I_a)^2}{(R_{day}-I_a+S)} Qsurf​=(Rday​−Ia​+S)(Rday​−Ia​)2​

    其中，QsurfQ_{surf}Qsurf​是累计径流或超额降雨（mm H2OH_2OH2​O），RdayR_{day}Rday​是第i天的降雨深度（mm H2OH_2OH2​O），IaI_aIa​是初始损失，包括径流之前的地表蓄水，截留和产流前的入渗（mm H2OH_2OH2​O），S是持蓄参数（mm H2OH_2OH2​O），该值在由于土壤类型、土地利用、管理措施和坡度的不同空间变化，以及土壤含水量的随时间的变化而变化。

    S=25.4(1000CN−10)S=25.4(\frac{1000}{CN}-10) S=25.4(CN1000​−10)

    其中，由于IaI_aIa​的初始值一般为0.2S，于是，方程可以变为：

    Qsurf=(Rday−0.2S)Rday+0.8SQ_{surf}=\frac{(R_{day}-0.2S)}{R_{day}+0.8S} Qsurf​=Rday​+0.8S(Rday​−0.2S)​

    只有当Rday>IaR_{day}>I_aRday​>Ia​时，径流才会产生。

    由此可见，SCS径流曲线数越大，相同降水条件下产生的径流量越大。
