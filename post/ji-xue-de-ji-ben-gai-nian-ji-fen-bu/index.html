积雪的基本概念及分布

2020-01-07 3 min read # Hydrology # Notes # Cryosphere

对《寒区水文导论》一书的第六章(积雪的基本概念及分布)中重点内容和概念的总结（读书笔记）
Citation: 丁永建, 张世强, 陈仁升, 等. 2017, 寒区水文导论, 科学出版社.

基本概念

  • 积雪(snow cover)
    地球表面存在时间不超过一年的雪层，即季节性积雪(seasonal snow cover)。
  • 雪深
    从积雪面到地面的垂直深度(cm)，又称积雪厚度，随积雪的加深不断累积变化。
  • 积雪密度
    地面积雪层中单位体积的质量(kg/m3或g/m3kg/m^3或g/m^3kg/m3或g/m3)，雪密度随时间发生变化，主要受地热、上面雪的压力、风、积雪内温度和水分的变化、融雪水的渗漏等的影响。
    积雪密度随积雪变质作用的进程、雪层液态水含量的增加、孔隙度减少而逐步增大，降雪密度逐步增大的这一过程，称为密实化过程：

    ρs=ρi(1−ϕ)+ρwϕSw\rho_{s}=\rho_i(1-\phi)+\rho_w\phi{S_w} ρs​=ρi​(1−ϕ)+ρw​ϕSw​

    其中，ρs\rho_{s}ρs​为积雪密度(kg/m3kg/m^3kg/m3)，ρi\rho_iρi​为冰的密度(917kg/m3917kg/m^3917kg/m3)，ρw\rho_wρw​为水的密度(103kg/m310^3kg/m^3103kg/m3)，SwS_wSw​为孔隙中的饱和体积含水率(m3/m3m^3/m^3m3/m3)，ϕ\phiϕ为积雪中的孔隙度(m3/m3m^3/m^3m3/m3)。
  • 雪水当量
    雪的当量水深度，是积雪层完全消融后得到的水形成的水层的垂直深度(mm)，可以直接测量，也可以通过积雪深度(ds,mmd_s, mmds​,mm)和密度(ρs,kg/m3\rho_s, kg/m^3ρs​,kg/m3)计算：

    SWE=0.01×ds×ρsSWE=0.01\times{d_s}\times{\rho_s} SWE=0.01×ds​×ρs​

  • 降雪强度
    单位时间的降雪量(mm/h)。
  • 热导率
    指的是积雪固体热传导和潜热传递的一个组合结果，其中冰晶的传导率一般随温度的升高而减小，但潜热传导则相反，总之，积雪的热传导会随雪温的升高而增加：

    q=−keffdT/dZ q=-k_{eff}dT/dZ q=−keff​dT/dZ

    式中，q为雪层热传导(W/m^2)，keffk_{eff}keff​为雪层有效的热导率[W/(m⋅K)W/(m\cdot{K})W/(m⋅K)]，dT/dZdT/dZdT/dZ为雪层中的温度梯度(K/m)。
  • 降雪条件
    • 大气要含有较冷的冰晶核
    • 充分的水汽
    • 足够低的气温

积雪的重要性

  • 积雪的高反照率极大地影响了地球表面的能量平衡，进而影响了区域和全球尺度的气候，气候系统的变化由反过来影响积雪的分布状态。
  • 积雪是重要的固态水库，在消融季节可以补给土壤、地下水、河流和水库。
