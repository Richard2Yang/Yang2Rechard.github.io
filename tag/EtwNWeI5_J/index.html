径流与流量
2019-12-31 3 min read # Hydrology # Notes

流域地表面的降水，如雨、雪等，沿流域的不同路径向河流、湖泊和海洋汇集的水流叫径流。在某一时段内通过河流某一过水断面的水量称为该断面的径流量。径流是水循环的主要环节，径流量是陆地上最重要的水文要素之一，是水量平衡的基本要素。
