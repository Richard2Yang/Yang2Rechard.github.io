Learning PyQt 5

  1. 2019-11-08
  2. 137字
  3. 1 min read

Env: Python 3, PyQt 5
Reason: Python 3默认使用UTF-8编码，不需要考虑中文编码的问题
Notes: PyQt 5.9之后不再提供常用的Qt工具, 如图形界面开发工具Qt Designer等, 需要另外安装(pip install PyQt5-tools)

Qt Designer

PyQt 5采用两种方式制作程序UI界面(UI制作工具+纯代码编写)
Qt Designer是PyQt 5制作UI界面的工具, 用于生成UI界面的.ui文件, 之后通过命令将.ui文件转换为.py文件, 并被其他Python文件引用。
